## Multiplication

+ 행렬의 표현
    + key(i, m) value (p, a) -> A 행렬의 i번째 행 p번째 열의 값은 a이다.
    + A 1 1 3 -> A 행렬의 1,1 value 는 3이다. 
+ 입력 파일
    + 형식 `행렬이름<tab>행번호<tab>열번호<tab>원소값`
    + 예시 `A 0 0 3`
+ 출력 파일
    + 형식 `행번호<tab>열번호<tab>원소값`
    + 예시 `0 0 1` (순서는 임의)
> key(i, m) value (p, a) 에서 m은 해당 a 값이 m번 계산되어야 하므로 (행렬 곱연산) 그 갯수만큼 생성한다.
